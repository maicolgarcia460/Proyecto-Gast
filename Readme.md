# Proyecto GAST
## Gestión y Asignación de Solicitudes de Trabajo

---

## Descripción del Proyecto

GAST es una aplicación web en desarrollo orientada a la gestión y asignación de solicitudes dentro del área de trabajo de una organización. El sistema busca facilitar el registro, asignación, aprobación y seguimiento de solicitudes, permitiendo además visualizar estadísticas relacionadas con el proceso de gestión.

El objetivo principal del sistema es mejorar la organización y control de las solicitudes de trabajo, optimizando la trazabilidad de cada solicitud desde su registro hasta su finalización.

Actualmente el proyecto se encuentra en fase de desarrollo, contando con un frontend funcional que representa la estructura visual del sistema.

---
